# ucidata 0.0.3

## Features

- Select datasets from the [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/index.php)
  as `data.frame` objects with appropriate type-casts.
- Accompanying documentation is available via `?dataset`.
